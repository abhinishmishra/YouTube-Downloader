# YoutTube - Downloader

A modern YouTube Video & Audio Downloader built using **Flask**, **Python**, and **yt-dlp**.
Download videos in different qualities or extract MP3 audio easily.

---

##  Features

✅ Download YouTube Videos
✅ Download MP3 Audio
✅ Select Video Quality
✅ Clean Responsive UI
✅ Fast Download Speed
✅ Flask Backend
✅ yt-dlp Integration

---

##  Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* yt-dlp

---

##  Project Structure

TubeFetch/
│── app.py
│── templates/
│── static/
│── downloads/

---

## Installation

```bash
pip install -r requirements.txt
python app.py
```

##  Run Project

Open browser:

```bash
http://127.0.0.1:5000
```
